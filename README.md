# Análise de Algoritmos II — Tradução PT-BR

Tradução para o **Português do Brasil** das notas de aula *Análise de Algoritmos II* de Mario Leston-Rey (junho de 2026).

## Como usar localmente

Abra `index.html` no navegador:

```bash
open index.html          # macOS
xdg-open index.html      # Linux
start index.html         # Windows
```

As fórmulas matemáticas são renderizadas via MathJax (requer conexão com a internet na primeira carga).

## Estrutura

```
index.html          ← Capa e sumário
capitulo1.html      ← Teoria básica de grafos
capitulo2.html      ← Algoritmos gulosos básicos
capitulo3.html      ← Árvores geradoras mínimas
assets/
  css/style.css     ← Folha de estilos
  figures/          ← Figuras extraídas do PDF
```
